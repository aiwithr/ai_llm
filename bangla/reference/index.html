    রেফারেন্স - বাংলা¶

    টেকনিক্যাল রেফারেন্স আর অতিরিক্ত তথ্য। এখানে প্রজেক্টের সব ডকুমেন্টেশন, বেস্ট প্র্যাকটিস, আর API রেফারেন্স আছে। নতুনদের জন্য গাইড আছে, আর অভিজ্ঞদের জন্য advanced টিপস আছে।

    রেফারেন্স ডকুমেন্টস¶

    রেফারেন্সে কিছু গুরুত্বপূর্ণ ডকুমেন্ট আছে। Reasoning Guide-এ reasoning কেন দরকার সেটা ব্যাখ্যা আছে। Best Practices-এ সঠিক পদ্ধতি আছে। API Documentation-এ এনডপয়েন্টের বিস্তারিত আছে। Troubleshooting-এ সমস্যা সমাধানের উপায় আছে।

    সহায়ক রিসোর্স¶

    সহায়ক রিসোর্সগুলো দরকারি। Reasoning Guide দেখলে বুঝবে কেন reasoning মেশিনের জন্য দরকার। API Reference দেখলে জানবে কোন এনডপয়েন্ট কিভাবে ব্যবহার করতে হয়। Troubleshooting দেখলে সমস্যার সমাধান পাবে।
